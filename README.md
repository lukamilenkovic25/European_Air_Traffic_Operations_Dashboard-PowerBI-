# European_Air_Traffic_Operations_Dashboard-PowerBI-
## Project Overview
This project presents an executive‑level Power BI dashboard focused on European airline operations performance.
The analysis covers flight volumes, cancellations, delays, on‑time performance, aircraft types, and regional operational drivers, with a clear separation between performance overview and root‑cause analysis.
The dataset is simulated but industry‑realistic, based on publicly available aggregated European aviation traffic statistics (e.g. EUROCONTROL‑style metrics), and is designed to showcase professional BI modeling and dashboard design.

## Business Questions
- How many scheduled flights operate across Europe over time?
- Which airlines and regions experience the highest cancellation rates?
- Where do delays originate: airline, aircraft type, or region?
- How strong is overall on‑time performance across major European carriers?

## Dataset Description
The dataset includes monthly airline‑level operational metrics:
- Scheduled and cancelled flights
- Cancellation rate
- Average delay minutes
- On‑time performance
- Aircraft type
- Regional distribution
- Estimated passenger capacity
